# Contributing

1. Fork the repo, create a branch: `git checkout -b feature/your-feature`
2. Install Rust + stellar-cli (see README)
3. `cd contracts/hello-world && cargo test`
4. Make your change, commit, push, open a PR
5. Link the issue your PR resolves

Look for issues labeled `good first issue` if you're new here.

## Code style

- One contract per folder under `contracts/`
- Include unit tests for any new contract method
- Document storage keys and any breaking changes to contract interfaces in your PR
